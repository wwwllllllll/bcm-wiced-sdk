wiced_result_t wiced_time_get_iso8601_time ( wiced_iso8601_time_t *  iso8601_time)

Get the current UTC time in iso 8601 format e.g.

"2012-07-02T17:12:34.567890Z"

Note
The time will roll over every 49.7 days
