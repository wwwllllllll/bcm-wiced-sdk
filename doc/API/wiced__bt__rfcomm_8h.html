enum wiced_bt_rfcomm_event_t

RFCOMM Event.

Enumerator
RFCOMM_EVENT_CONNECTION_ERROR 

RFCOMM connection error has occurred.

RFCOMM_EVENT_CONNECTION_ACCEPTED 

RFCOMM connection request from a remote device has been accepted.

RFCOMM_EVENT_CONNECTION_DISCONNECTED 

RFCOMM connection has been disconnected by the remote device.

RFCOMM_EVENT_INCOMING_PACKET_READY 

Incoming packet is ready.

Call wiced_bt_rfcomm_receive_packet to retrieve packet

RFCOMM_EVENT_CLIENT_CONNECTED 

RFCOMM client is connected to a remote device.
