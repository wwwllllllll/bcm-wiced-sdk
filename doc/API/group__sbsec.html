Force a SmartBridge socket to initiate Pairing Request with a Bluetooth Smart device.

Note
This function prepares the socket to initiate Pairing Request with the device. Previously loaded bond information is deleted and is replaced with a new bond if the pairing process succeeds.
Warning
  • This function must be called before calling wiced_bt_smartbridge_connect
  • Callback functions run on the context of WICED_NETWORKING_WORKER_THREAD
Parameters
[in,out]socket: pointer to the socket to enable pairing
[in]settings: security settings
[in]numeric_passkey: NUL-terminated string of numeric passkey. Maximum is 6 digits.
[in]bondingcallback : callback that is called when bonding with remote device occurs

Set a peer device's bond information.

Note
This function loads Bond (security) information to the socket which will be used to reestablish an encrypted link with the peer device without having to perform the complete handshake procedure (pairing)
Warning
This function must be called before calling wiced_bt_smartbridge_connect
Parameters
[in,out]socket: socket to load the Bond information into
[in]settings: security settings
[in]bond_info: Bond information
