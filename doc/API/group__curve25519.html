Calculate a public (shared) key given a basepoint and secret key.

Parameters
mypublic_outputReceives the 32 byte output shared key
secretThe 32 byte secret key. Must have been randomly generated then have the following operations performed on it: secret[0] &= 248; secret[31] &= 127; secret[31] |= 64;
basepointThe starting point for the calculation - usually the public key of the other party
Returns
0 when successful
